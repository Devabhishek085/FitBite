# 🍽️ FitBite — Smart Calorie Tracker

A modern, aesthetic calorie tracking web app built with React that helps users monitor their daily nutrition with a beautiful UI and smart features.

---

## ✨ Features

* 🔥 Add, edit, delete food items
* ⚡ Auto-fill nutrition using USDA API
* 🎯 Daily calorie goal tracking
* 📊 Animated calorie progress ring
* 🧊 Smart Plate UI (macro distribution visualization)
* 🔥 Streak system (gamification)
* 🌙 Dark / Light mode toggle
* 💾 Data stored in localStorage
* 🎨 3D animated background UI

---

## 🧠 Auto Nutrition Feature

Users can simply enter a food name and click **"Auto Fill"** to fetch:

* Calories
* Protein
* Carbohydrates
* Fat

Powered by **USDA FoodData Central API**

---

## 🛠️ Tech Stack

* React (Vite)
* Tailwind CSS
* Framer Motion
* JavaScript (ES6+)
* USDA FoodData Central API

---

## 📦 Installation

```bash
git clone https://github.com/Devabhishek085/FitBite.git
cd FitBite
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in root:

```env
VITE_USDA_API_KEY=your_api_key_here
```
